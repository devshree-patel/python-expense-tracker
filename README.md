# python-expense-tracker
A command-line based expense tracker built using Python.

## Features
- Add multiple expenses until user exits
- Categorize expenses
- Store data using CSV files
- Track monthly budget
- Calculate daily budget recommendation
- Handle invalid user input gracefully

## Technologies Used
- Python
- CSV file handling

## How to Run
1. Clone the repository
2. Run `expense_tracker.py`
3. Enter expenses as prompted
